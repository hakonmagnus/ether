# Contributing to Ether

The following is a set of guidelines for contributing to Ether. These are mostly guidelines, not rules. Feel free to propose changes to this document in a pull request.

## Code of Conduct

Please check our [Code of Conduct](/CODE_OF_CONDUCT.md) before contributing.

## Reporting Bugs

When creating a bug report, include as many details as possible and fill out [the required template](/.github/ISSUE_TEMPLATE/bug_report.md).

## Pull Requests

When creating a pull request, please follow the instructions in [the template](/PULL_REQUEST_TEMPLATE.md).
